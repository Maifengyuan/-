### 1.1 网站建设目标
- 网站围绕非物质文化遗产、环境保护为建设好发展方向，对相关内容进行宣传

- 网站面向全球群体，试图通过互联网将人类文化的珍贵遗产进行宣传，提升我们对期的保护意识
